# API Reference

Welcome to the EthoGraph API reference. This section provides detailed documentation for all public modules, classes, and functions, grouped by functionality. Click a module name to view its full API documentation.

| Module | Description |
|--------|-------------|
| [TrialTree](trialtree.md) | Core data container for multi-trial behavioral datasets |
| [Label Intervals](label-intervals.md) | Interval-based label creation, editing, and conversion |
| [Data Utilities](xr-utils.md) | xarray selection helpers and DataFrame export |
| [Neural](features/neural.md) | Firing rate computation and PCA from spike data |
| [Changepoints](features/changepoints.md) | Peak/trough detection and label boundary correction |
| [Movement](features/movement.md) | Kinematics, distances, angles, video motion energy |
| [Energy Envelopes](features/energy.md) | Amplitude envelope extraction for signals |
| [Preprocessing](features/preprocessing.md) | Signal resampling, smoothing, and normalization |
| [Audio Changepoints](features/audio-changepoints.md) | Vocal onset/offset detection from audio |
| [Oscillatory Events](features/oscillatory.md) | Rhythmic burst detection (theta, ripples, spindles) |
| [Validation](validation.md) | Dataset and TrialTree validation helpers |
